Qual é o futuro da internet?

A Inteligência Artificial torna-se uma área cada vez mais mediática e impactante no nosso quotidiano. O seu crescimento tem vindo a aumentar consideravelmente e prevê-se que assim continue nos próximos anos.

Machine learning é um campo da Inteligência Artificial, sendo uma área tecnológica multi-disciplinar que permite que computadores realizem tarefas específicas sem serem explicitamente programados, tomando decisões autónomas através da análise de padrões de dados.

Neste workshop vai ser mostrado como abordar um problema, através de metodologias de machine learning, colocando em prática o processo standard de criação de previsões.

Serão abordados os conceitos estruturais e mais básicos que possibilitam a utilização de inteligência artificial para resolver problemas reais numa sessão essencialmente prática.


Requisitos Mínimos

  • Conhecimentos básicos de programação
  • Um IDE de python ( preferencialmente o Pycharm )
  • Instalar o Anaconda 3.7
  • Instalar a biblioteca de python Scikit-learn
